Detailed Description

Dummy example header that mimics the macro parameters passed to the CL compiler.

In reality these macros are set in the Python wrapper files, e.g., integration.py, in the variable compile_options, itself set by pocl.set_compile_options(). This example reflects the set of -D macro flags passed when building the kernel integrate_trajectory() in integration.cl.
